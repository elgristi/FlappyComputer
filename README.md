# Flappy Computer

*By [Gristlymm24](https://github.com/elgristi)*, readme.md by [*hacker-3342*](https://github.com/hacker-3342)

Flappy Computer is a PC game based on the popular [Flappy Bird](https://en.wikipedia.org/wiki/Flappy_Bird) mobile game launched back in 2013. The game is written fully in C++, and includes shell scripts for installation.

## How to run

To download and play the game, head over to the [GitHub releases page](https://github.com/elgristi/FlappyComputer/releases) and select the latest stable build for your system.

## Compiling from source

To compile the project from source, you will need to clone the repository using [git](https://git-scm.com/) or download the .zip file from GitHub.

You will also need a C++ compiler like [GCC](https://gcc.gnu.org/), and the [SFML library](https://www.sfml-dev.org/). Then, run either `compile_linux.sh` for Linux, and `compile_win.sh` for Windows.

## Features

The game features lots of interesting and fun to play mechanics and media, such as:

- Simple mechanics (click-to-jump)
- Creative pixel art for the assets and texts
- Catchy copyright-free music & sound effects
- High score persistent tracking system
- And more to come in future versions!
